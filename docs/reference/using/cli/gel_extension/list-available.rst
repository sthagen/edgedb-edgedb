============================
gel extension list-available
============================

List available extensions.

.. cli:synopsis::

  gel extension list-available [<options>]

Options
=======

See :ref:`Connection options <ref_cli_gel_connopts>` for global options.

Examples
========

.. code-block:: bash

  $ gel extension list-available
  ┌─────────┬───────────────┐
